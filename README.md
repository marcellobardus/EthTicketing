# EthTicketing

## Usage 

### Clone repository

`bash git clone https://github.com/spaghetticoderIT/EthTicketing.git`

### Go to the created directory

`bash cd EthTicketing`

### Install truffle dependencies

npm: 

`bash npm install`

yarn:

`yarn install`

### Install cli dependencies

`bash cd cli`

`bash yarn install`
